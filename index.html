Exercice 1

Soit le processus stochastique \left\{ X_t \right\}_{t \in \mathbb N} défini par X_0 = 0 et X_t = \tg^{-1} \left( X_{t - 1} + m p Y_t - m p^2 \right) où les Y_t sont des variables aléatoires Binomiales(m, p) indépendantes. Implémentez les fonctions suivantes:

  1. w_for(n, m = 1, p = 0.5) qui simule une trajectoire de longueur n de ce processus en utilisant une boucle for;

  2. w_vec(n, m = 1, p = 0.5) qui simule une trajectoire de longueur n de ce processus sans utiliser de boucle for.

Exercice 2

Pour un ensemble \bm{x} = \left\{ x_1, x_2, \ldots \right\}, soit \bm{x}_{\alpha} = \{ x \in \bm{x} : x > q_{\alpha}(\bm{x}) \} où q_{\alpha}(\bm{x}) est le quantile d’ordre \alpha de \bm{x}. On définit la statistique T_{\alpha}(\bm{x}) = \frac{1}{| \bm{x}_{\alpha} |} \sum_{\{ x \in \bm{x}_{\alpha} \}} x

Implémentez la fonction ex2(x, alphas = seq(0, 0.99, 0.01)) qui calcule cette statistique pour un ensemble de \alpha donné.

Pour référence, ex2 appliquée aux exemples précédents.

Exercice 3

Soit la famille de vecteurs définie par \bm{x}_t = ((t \!\!\!\!\mod 2) + 1) \bm{x}_{t - 1} {\bm{x}}^\intercal_{t - 1} \bm{v} \mod 8191 pour un vecteur \bm{v} compatible donné. On veut générer un (mauvais ?) nombre aléatoire entre 0 et 1 de la manière suivante:

  1. choisir n entiers x_1, \ldots x_n et définir \bm{x}_0 = {(x_1 \cdots x_n)}^\intercal;

  2. choisir 1 entier v_1 et définir \bm{v} = {(v_1\, (v_1 + 1)\, \cdots (v_1 + n - 1))}^\intercal;

  3. appliquer la récurrence ci-haut 1000 fois;

  4. retourner \frac{{\bm{x}}^\intercal_{1000} \bm{x}_{1000} \mod 100}{100}.

Implémentez cet algorithme dans une fonction nommée cheaprng(..., v1 = 1).

Quelques valeurs:

  • \bm{x}_0 = (10, 2, 45, 54, 55, 1000, 12, 18, 5), v_1 = 1 \mapsto 0.24

  • \bm{x}_0 = (10, 2, 45, 54, 55, 1000, 12, 18, 5), v_1 = 1234 \mapsto 0.25

  • \bm{x}_0 = (10), v_1 = 1 \mapsto 0.00

  • \bm{x}_0 = (10, 18), v_1 = 1 \mapsto 0.54
